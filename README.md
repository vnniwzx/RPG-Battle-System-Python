Sistema simples de RPG em Python desenvolvido para praticar lógica de programação, estruturas condicionais, loops e Programação Orientada a Objetos (POO).

🚀 Funcionalidades
Sistema de batalha em turnos
Ataque e defesa
Vida dos personagens
Inimigos aleatórios
Sistema de dano
Interface via terminal

🛠️ Tecnologias utilizadas
Python

📚 Objetivo do projeto
Este projeto foi desenvolvido com foco em aprendizado e prática de programação utilizando Python, explorando conceitos fundamentais de desenvolvimento de sistemas e lógica de programação.

▶️ Como executar
Clone o repositório:
git clone https://github.com/vnniwzx/nome-do-repositorio.git
Acesse a pasta do projeto:
cd RPG-Battle-System-Python
Execute o arquivo: python rpg.py
